# Weather Plus #

* 작성자: Adriano Barbieri
* 다운로드: [안정 버전][1]

# Weather Plus 소개: #

* 이 추가 기능은 NVDA에서 현지 기온, 24시간 예보, 최대 2일 추가 예보를 제공합니다.
* 저작권 (C) [Adriano Barbieri](mailto:adrianobarb@yahoo.it)
* GNU GPL(General Public License) v2에 따라 배포됩니다.
* Weather Plus는 다음 서비스를 사용하여 작동합니다.
* [https://www.weatherapi.com/](https://www.weatherapi.com/)
* [http://www.geonames.org/](http://www.geonames.org/)
* [http://veloroutes.org/elevation/](http://veloroutes.org/elevation/)
* [https://www.nvda.it/](https://www.nvda.it/)

# 사용법: #

* 현재 기온과 날씨 상태 정보를 확인하려면 NVDA+w를 누르십시오.
* 24시간 예보와 최대 2일 예보를 확인하려면 NVDA+shift+W를 누르십시오.
* 임시 도시를 설정하려면 NVDA+shift+control+w를 누르십시오.
* Weather Plus 설정 대화상자를 열려면 NVDA+shift+control+alt+w를 누르십시오.
* 날씨 보고가 업데이트된 날짜와 시간을 확인하려면 NVDA+alt+w를 누르십시오.
* 화씨, 섭씨, 켈빈 온도 단위를 전환하려면 control+shift+w를 누르십시오.

# Weather Plus 설정: #

* 처음 사용하기 전에 Weather Plus 추가 기능을 설정해야 합니다! 환경 설정 하위 메뉴의 Weather Plus 설정 하위 메뉴로 이동한 뒤 다음 옵션 중 하나를 선택하십시오.
 * 도시 설정 및 관리... - 목록에서 현재 도시를 표시하거나 설정합니다.
 * 임시 도시 설정... - 가능한 경우 목록에서 임시 도시 하나를 표시하고 설정할 수 있습니다.
 * 문서 - 현재 언어의 도움말 파일을 엽니다.
 * 업데이트 확인... - 새 버전이 있는지 알려줍니다.

새 도시를 추가하려면 다음 항목을 누르십시오.

* 도시 설정 및 관리... - 목록에서 현재 도시를 표시하거나 설정합니다.
* 다음 메시지는 처음 한 번만 표시됩니다! 설정 프리셋 없음 F1: 위치 도움말, F2: 마지막 TAB 선택, F3: 목록 및 편집 상자, F4: 날씨 예보 기간 조절, F5: 볼륨 조절.
* 편집 상자에 도시를 입력하거나, 가능한 경우 목록에서 하나를 선택하십시오. 참고: F5 키는 효과음이 활성화되어 있을 때 사용할 수 있습니다.
* "도시 설정 및 관리..." 항목에서 Enter를 누르면 다음 버튼들이 표시됩니다.
* 테스트 - 도시가 유효한지 테스트하고 해당 데이터를 찾습니다.
* 추가 - 현재 도시를 목록에 추가합니다. 입력한 도시가 테스트를 통과했거나 목록에서 도시를 선택한 경우 이 버튼을 사용할 수 있습니다.
* 세부 정보 - 현재 도시 정보를 표시합니다. 목록에서 도시를 선택했거나 도시가 테스트를 통과한 경우 이 버튼이 활성화됩니다.
* 정의 - 효과음을 맞추기 위해 지역 유형을 정의할 수 있습니다. 오디오 효과가 설치 및 활성화되어 있고 목록에서 도시를 선택한 경우 이 버튼을 사용할 수 있습니다.
* 프리셋 - 도시를 기본값으로 설정하며, 플러그인을 다시 시작할 때마다 사용됩니다. 목록에 이전에 추가되었지만 프리셋으로 지정되지 않은 도시를 선택했거나 도시가 테스트를 통과한 경우 이 버튼이 활성화됩니다.
* 제거 - 목록에서 현재 도시를 삭제합니다. 목록에 이전에 추가된 도시를 선택한 경우 이 버튼을 사용할 수 있습니다.
* 이름 변경 - 현재 도시의 이름을 변경합니다. 목록에 이전에 추가된 도시를 선택한 경우 이 버튼을 사용할 수 있습니다.
* 새 도시 가져오기... - *.zipcodes 확장자를 가진 다른 도시 목록에서 도시를 가져올 수 있습니다. 연결된 확인란을 선택하여 가져올 도시를 고를 수 있습니다.
* 도시 내보내기... - 도시를 *.zipcodes 확장자의 지정된 파일에 저장할 수 있습니다. 목록에 도시를 하나 이상 추가하고 저장한 경우 이 버튼이 활성화됩니다.
* 온도 단위: 라디오 버튼을 사용하여 섭씨(기본값), 화씨, 켈빈 중에서 선택합니다.
* 온도 표시: 라디오 버튼을 사용하여 Celsius `-` Fahrenheit `-` Kelvin(기본값), C `-` F `-` K, 지정 안 함 중에서 선택합니다.
* 콤보 상자: 날씨 예보 일수: 1; 1부터 3까지 선택할 수 있습니다(기본값은 1일).
* 콤보 상자: API 응답 언어: English, en; 날씨 상태 텍스트의 언어를 선택할 수 있습니다.
* 다음 작업을 수행하려면 해당 확인란을 전환하십시오.
* 도시 세부 정보를 포함한 날씨 보고와 날씨 예보를 클립보드에 복사; 확인란 선택 안 됨(기본값)
* 오디오 효과 사용(현재 날씨 상태에만 적용); 이 확인란으로 효과음 설치도 관리할 수 있습니다. 효과음이 설치되어 있고 확인란이 활성화되어 있으면 F5 키와 볼륨 설정을 사용할 수 있습니다.
* 추가 확인란인 "날씨 효과만 사용"도 사용할 수 있습니다.
* 전체 볼륨을 변경하거나 마지막으로 들은 효과음을 조절하고 주변 환경음을 걸러낼 수 있습니다. 기본값은 선택 안 됨입니다.
* 날씨 효과만 사용 - 효과음이 활성화된 경우 사용할 수 있는 옵션입니다. 활성화하면 비, 바람, 천둥 등 날씨 효과만 들을 수 있고 모든 환경음은 걸러집니다. 기본값은 선택 안 됨입니다.
* 24시간 형식으로 시간 읽기 사용 - 이 확인란을 선택하지 않으면 예를 들어 12 AM `-` 12 PM처럼 12시간 형식으로 시간을 알립니다. 기본값은 선택됨입니다.
* 설정 창에 도움말 버튼 사용; 확인란 선택됨(기본값)
* 바람 정보 읽기; 확인란 선택 안 됨(기본값). 이 확인란을 활성화하면 다음 확인란도 활성화할 수 있습니다.
* 풍향 추가; 바람이 불어오는 방향을 나타냅니다. 확인란 선택됨(기본값)
* 풍속 추가; 킬로미터/시 또는 마일/시 단위의 속도를 나타냅니다. 확인란 선택됨(기본값)
* 풍속을 미터/초로 추가; 확인란 선택됨(기본값)
* 돌풍 속도 추가; 확인란 선택됨(기본값)
* 체감 온도 추가; 확인란 선택됨(기본값)
* 대기 정보 읽기; 확인란 선택 안 됨(기본값). 활성화하면 다음 확인란도 선택할 수 있습니다.
* 습도 값 추가; 습도를 백분율로 나타냅니다. 확인란 선택됨(기본값)
* 가시거리 값 추가; 볼 수 있는 거리를 킬로미터 또는 마일 단위로 나타냅니다. 확인란 선택됨(기본값)
* 기압 값 추가; 기압을 밀리바 또는 수은주 인치로 나타냅니다. 선택하면 기압을 수은주 밀리미터로 표시할 수 있는 추가 확인란이 활성화됩니다. 확인란 선택됨(기본값)
* 자외선 지수 값 추가; 확인란 선택됨(기본값)
* 천문 정보 읽기; 일출과 일몰 시각, 월출과 월몰 시각을 나타냅니다. 확인란 선택 안 됨(기본값)
* 소수 구분에 쉼표 사용; 활성화하면 쉼표를 소수 구분자로 사용하고, 비활성화하면 마침표를 사용합니다. 확인란 선택 안 됨(기본값)
* 업데이트 확인; 활성화하면 추가 기능 업데이트가 있을 때 알려줍니다. 확인란 선택됨(기본값)
* 작업을 확인하려면 확인 버튼을 누르고, 취소하려면 취소 버튼을 누르십시오.
* 도시 목록을 수정한 상태에서 "취소"를 누르면 알림이 표시되며, 그래도 저장할 수 있습니다. 참고: 설정은 다음 이름의 파일에 저장됩니다.
* "Weather.ini": Weather Plus 시작 설정.
* "Weather.volumes": 전체 볼륨과 별개로 저장되는 사용자 지정 오디오 볼륨 수준.
* "Weather.zipcodes": 도시, 우편번호, 정의 목록.
* "Weather_searchkey": 저장된 검색 키.
